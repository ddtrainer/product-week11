
# Lotto Number Generator

## Overview

This application will be a simple and visually appealing Lotto Number Generator. It will allow users to generate a set of unique lottery numbers with a single click. The application will be built using modern web standards, including HTML, CSS, and JavaScript, and will feature a clean and responsive design.

## Features

*   **Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Display:** Displays the generated numbers in a clear and attractive format.
*   **User Interface:** A simple and intuitive interface with a button to trigger the number generation.
*   **Styling:** Modern and visually appealing design.
